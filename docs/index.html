BoxGrid

BoxGrid is a flexbox based grid system. Using flexbox we can gain several advantages over older grids. We can include features such as equal height columns, and column re-ordering.

The grid has been tested in the lastest Firefox, Chrome, Safari, IE11, 10 and 9 (although JS is required for IE9)

The Grid

The grid is based on a maximum of 12 columns. A containing element must be placed around the columns with a class of boxgrid.

Responsive Break-points

The grid uses 5 break-points to allow for varying layouts at different screen sizes. Apply the prefix followed by the number of column span required, to each column.

Styles for each break-point override the previous.

Name Class prefix Screen size
X-Small xs_ >= 300px
Small sm_ >= 450px
Medium md_ >= 750px
Large lg_ >= 1200px
X-Large xl_ >= 1600px

Usage

For all modern browsers (IE 10+, Firefox 28+, Chrome 21+, Safari 6.1+, Opera 12.1+) just include the boxgrid.css stylesheet.

For IE9 (and possibly below, although it hasn't been tested), include the boxgrid-ie.css stylesheet, the boxgrid-ie-fix.js file and a copy of jQuery (we use 1.11.2 although others should be fine) inside a conditional tag

Source code:

<link href="../css/boxgrid.css" rel="stylesheet" type="text/css">
<!--[if IE 9]>
<link href="../css/boxgrid-ie.css" rel="stylesheet" type="text/css">
<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.2/jquery.min.js"></script>
<script type="text/javascript" src="../js/boxgrid-ie-fix.js"></script>
<![endif]-->

Column sizes

All columns must have a class of col. By default a column will be 100% wide.

Apply a prefixed number to specify how many columns a particular column will span e.g. md_4 (medium span 4).

Column sizes range from 0 (zero) to 12 (twelve).

To use pre-spacing, add a class constructed of the break-point prefix along with a prefix of pre_ followed by the number of columns spacing required.

Post column spacing

You may wish to prevent any further spacing after a column from being used. This is particularly useful if you wish to create the effect of a new row.

To apply post-spacing, add a class constructed of the break-point prefix along with a prefix of post_ followed by the number of columns space required.

8
4
3

Source code:

<div class="boxgrid">
<div class="col xs_8 xs_post_4">8</div>
<div class="col xs_4 xs_post_8">4</div>
<div class="col xs_3">3</div>
</div>

Changing column order

You can change the order of columns using data attributes, keeping the HTML in the order you want but simply swapping the columns around using styles.

Add a data attribute of data-bg_xs_order="2" to the HTML element (replacing xs_ with the relevant break-point prefix and the 2 with the order number required). You can use an order number of 1 through to 12.

One
Two
Three
Four

Source code:

<div class="boxgrid">
<div class="col xs_3" data-bg_xs_order="4" data-bg_sm_order="2" data-bg_md_order="3" data-bg_lg_order="4" data-bg_xl_order="4">
One
</div>
<div class="col xs_3" data-bg_xs_order="3" data-bg_sm_order="3" data-bg_md_order="4" data-bg_lg_order="1" data-bg_xl_order="2">
Two
</div>
<div class="col xs_3" data-bg_xs_order="2" data-bg_sm_order="4" data-bg_md_order="1" data-bg_lg_order="2" data-bg_xl_order="3">
Three
</div>
<div class="col xs_3" data-bg_xs_order="1" data-bg_sm_order="1" data-bg_md_order="2" data-bg_lg_order="3" data-bg_xl_order="1">
Four
</div>
</div>

Content alignment

You can change the horizontal and vertical alignment of content within a column, and this can differ per breakpoint.

Horizontal alignment

To apply horiztonal alignment apply a class built up of break-point prefix followed by left, center or right.

Left aligned
Centre aligned
Right aligned
<div class="boxgrid">
<div class="col xs_12 xs_left">Left aligned</div>
<div class="col xs_12 xs_center">Centre aligned</div>
<div class="col xs_12 xs_right">Right aligned</div>
</div>

Vertical alignment

To apply vertical alignment apply a class built up of break-point prefix followed by top, middle or bottom.

Top
Middle
Bottom
<div class="boxgrid">
<div class="col xs_4 xs_top" style="height: 80px;">Top</div>
<div class="col xs_4 xs_middle">Middle</div>
<div class="col xs_4 xs_bottom">Bottom</div>
</div>
